# MyRAM

MyRAM catches what your brain drops, then helps you make sense of it later.

MyRAM is a personal notes app for quickly saving notes, screenshots, pasted text, and fragments before they get lost. It is built around fast capture first, with organization and review available when there is more time and context.

## Features

- Create and edit notes
- Use mobile editing controls that stay close at hand
- Attach pictures to notes
- Copy and paste text from pictures using OCR
- Export and share notes
- Pin important text inside a note for easier reference

## Platform

This repository contains the Android implementation of MyRAM.

An iOS implementation is maintained separately.
